# Local font files

Drop the Satoshi variable font files here (download from https://www.fontshare.com/fonts/satoshi):

- `Satoshi-Variable.woff2` — upright, weight range 300–900
- `Satoshi-VariableItalic.woff2` — italic, weight range 300–900

Both files are referenced by `src/app/fonts.ts`. Next.js will fail to build until they exist.
